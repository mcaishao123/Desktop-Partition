# 🌟 桌面整理工具 (Desktop Organizer Fences)

这是一个基于 **WPF + .NET 9.0** 开发的极轻量、高性能、高颜值 Windows 桌面分区整理工具。

通过深度调用 Windows 底层 Win32 API，本项目实现了类似于商业整理软件（如 Fences）的**原生壁纸级钉入、跨虚拟桌面常驻、以及文件拖拽收纳**功能，旨在为您提供最纯净、最流畅的桌面整理体验。

---

## ✨ 核心特色

*   **📺 原生壁纸级贴合（WorkerW 钉入）**：
    窗口的 Win32 Owner 物理指向系统桌面最底层 `WorkerW`（壁纸与图标渲染容器）。按 `Win + D` 显示桌面时**绝对不会被最小化隐藏**，与系统壁纸完美融为一体。
*   **💻 100% 虚拟桌面共享（零 CPU 占用）**：
    由于卡片窗口的 Owner 钉入了桌底，Windows 系统内核会自动、零延迟地在您新建或切换的**每一个虚拟桌面上同时渲染该分区**，完全无需定时器轮询，CPU 占用率常年为零。
*   **📂 智能文件拖拽收纳（OLE 拖放兼容）**：
    完美支持将桌面文件、文件夹直接拖入分区进行物理收纳。移入分区的物理文件会被自动移动到专属的分区归档夹下，删除分区时文件会自动完璧归赵放回系统桌面。
*   **✨ 双重工作模式（编辑/锁定模式）**：
    *   **开启编辑模式**：允许通过鼠标拖拽卡片边缘来**自由拉伸调整高宽大小**、双击分区标题重命名、以及拖动分区位置。
    *   **锁定模式（非编辑模式）**：屏蔽拉伸与双击标题，锁定卡片位置与大小，防止日常使用中发生误触。
*   **⚡ 快速专注模式**：
    双击系统托盘图标即可一键**显示/隐藏所有分区卡片**，让您在杂乱的临时文件和清爽的壁纸之间瞬间切换。
*   **💎 现代高颜值毛玻璃（Acrylic 亚克力模糊）**：
    深度集成 Windows 11/10 的系统级高斯模糊（Blur Behind）效果，支持动态磨砂和半透明白描边，并自带圆角，颜值拉满。
*   **🛡️ 双重防灾备份**：
    每次程序启动时，都会自动将您系统桌面上所有的文件和图标在当前运行目录下物理冷复制一份作为备份，防止任何意外发生。
*   **📦 绿色单文件发布 & 自动铺设**：
    支持自包含运行时的单文件 `.exe` 发布。首次运行后**自动在桌面上铺设带精美 Logo 的快捷方式启动图标**，支持托盘一键开启**开机自启动**。

## 🛠️ 技术选型

*   **核心框架**：WPF (.NET 9.0 - Windows Window Application)
*   **系统辅助**：Windows Forms (用于系统托盘 NotifyIcon 深度兼容)
*   **系统调用**：Win32 API P/Invoke (`user32.dll`, `dwmapi.dll`)
*   **资源嵌入**：自适应 32/64 位环境下的 PE 图标段资源物理注入 (`logo.ico`)

## 🚀 编译与发布打包指南

### 1. 本地调试开发
在控制台执行以下命令即可启动调试版：
```powershell
dotnet run --project .\桌面整理工具.csproj
```

### 2. 发布绿色单文件 .exe 包
运行以下官方打包命令，将所有代码、静态资源、以及整个 .NET 运行时打包到单个体积压缩的 `.exe` 文件中：
```powershell
# 彻底清空编译缓存，避免图标缓存干扰
Remove-Item -Recurse -Force bin, obj -ErrorAction SilentlyContinue

# 执行自包含单文件发布编译
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:PublishReadyToRun=true
```

*   **打包输出路径**：`bin\Release\net9.0-windows\win-x64\publish\桌面整理工具.exe`
*   **使用方式**：双击该 exe 即可运行。运行后会在您的桌面上自动派生启动图标。您可以随意将该单个文件拷贝或分发到其他未安装 .NET 的 Windows 设备上使用。

## ⚙️ 配置文件说明

所有的分区位置、自定义拉伸大小、文件排列顺序、以及全局设置均保存在本地：
`%AppData%\桌面整理工具\settings.json`

分区的归档物理路径保存在：
`%AppData%\桌面整理工具\Partitions\<Partition_Guid>\`
